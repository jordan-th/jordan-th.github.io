I recently built Guardrail, an open source, "traffic replay" microservice testing tool.

Guardrail is an open-source tool that generates regression tests for microservices using recorded production traffic.

Guardrail allows developers to verify the behavior of their microservice without manually writing test requests or setting up complicated end-to-end testing environments.

The UI allows developers to easily understand how changes to their microservice affect how it handles traffic.
